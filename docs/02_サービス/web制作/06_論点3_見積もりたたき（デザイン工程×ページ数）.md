# 06. 論点3：見積もりたたき（2エンジン構造）v0.2

> 2026/6/11 改訂。v0.1（フラットなページ数×単価）の穴を修正。
> 穴：①トンマナ¥15kは"考えることが無い低ビジネスレベル客"でしか成立しない②ページ数リニアは大規模で破綻
> ③Lv「参考あり/なし」は浅い。→ **価格を2エンジンに分離**して解く。
> 検証：KADODE実見積もり ¥420,000 がすでにこの構造（上流¥190k＋下流¥230k／47都道府県はテンプレ展開）。

---

## コア：価格は2つのエンジンの合算

```
Web総額 = 【A 上流：思考エンジン】 + 【B 下流：生産エンジン】 ＋ 実装(既存px式) ＋ 運用/保守
```
- **A 思考エンジン**：ビジネスの複雑さ＝**思考難度Lv**で決まる。**ページ数と無関係**。
- **B 生産エンジン**：ページ"型"の作り込み＋量産。**ページ数で効くが、テンプレ展開で非線形**。

> v0.1の失敗＝トンマナ（本来A）をB（ページ生産単価表）に置いた。だから¥15kに見えた。

---

## A. 上流：思考エンジン（ビジネスレベル＝思考難度Lvで価格）

含むもの：事業・市場理解(Bz)／ブランド・訴求設計(Br)／情報設計IA全体／**ビジュアル要件(トンマナ)**。
価格は `03_…` のエントリー層＝思考難度Lv（Lv1模倣〜Lv5文脈創出）に連動。**ページ数では動かない**。

| 思考難度Lv | クライアントのビジネスレベル | 上流でやること | 価格レンジ目安 |
| --- | --- | --- | --- |
| Lv1 | 低（考えることがほぼ無い） | 既存方針のトンマナ整理のみ | ¥15,000〜 |
| Lv2 | 中 | ペルソナ/3C仮説＋トンマナ再設計 | ¥40,000〜80,000 |
| Lv3 | 高 | 競合分析・訴求設計・IA全体設計 | ¥80,000〜150,000 |
| Lv4 | 概念抽出（理念/世界観の具現化） | ブランド再設計・VI整理 | ¥150,000〜 |
| Lv5 | 文脈創出（事業から設計） | 事業・収益構造から共創 | 要見積（最上位） |

> ここが「トンマナ¥15k問題」の答え：**¥15kはLv1（低ビジネスレベル）専用**。本格事業はLv3-4で数十万になる。
> ＝論点0のCONVERT（上流モジュール）の価格そのもの。エントリー層が上流なほど高い。

---

## B. 下流：生産エンジン（ページ"型" × 作り込みLv ＋ 量産）

ページを**「型(テンプレ)」と「量産(同型の流し込み)」に分ける**（リニア積み上げをやめる）。

```
生産費 = Σ_型 ( 情報整理 + 設計 + デザイン ) × 型のLv  +  量産ページ数 × 展開単価
```

### 型あたり単価（①情報整理＋②設計＋④デザイン）
| Lv | Lv1 | Lv2 | Lv3 |
| --- | --- | --- | --- |
| 型計 | ¥35,000 | ¥55,000 | ¥85,000 |

### 量産（同型の2ページ目以降）
- 同じ型を流し込むだけ → **型単価の約30%／ページ**（デザイン微調整＋流し込み。目安 ¥10,000〜/p）。
- ＝「47都道府県ページ」のような大量同型を、リニアで爆発させず吸収する。

### デザイン作り込みLvの真のドライバー（"参考あり/なし"を超える）
| 軸 | Lv低 ← → Lv高 |
| --- | --- |
| 創出量 | 型踏襲・参考あり ← → ゼロベース創出 |
| 訴求・演出密度 | 情報配置のみ ← → 訴求設計・アニメ・感情演出 |
| 情報量 | 少 ← → 多・図解 |
> 3軸の合算でLvを当てる。参考有無は「創出量」の一面にすぎない。

---

## たたき式（全体）

```
Web総額 = A思考エンジン(思考難度Lv) 
        + B生産エンジン( Σ型×Lv + 量産数×展開単価 )
        + 実装(px式) + 運用/保守
        （＋特急1.3 / 調整費2,000円/日）
```

### 検証：KADODE（実見積もり ¥420,000）にあてる
| 区分 | 実見積もり | 本式での解釈 |
| --- | --- | --- |
| CVR戦略コンサル ¥105,000 | A思考 | 競合/導線分析＝思考難度Lv3-4 |
| サイト設計 ¥85,000 | A思考 | IA再整理・トンマナ・CMS方針＝Lv3 |
| デザイン ¥90,000 | B生産 | TOP＋FV＋下層共通＝**型の設計**（Lv3型 数本） |
| 構築 ¥140,000 | 実装 | 47都道府県を**テンプレ展開**で吸収（リニアなら爆発） |
→ **上流¥190k＋下流¥230k**。実績が2エンジン構造とテンプレ展開を裏付ける。◎

---

## この改訂で解けたこと
- トンマナ¥15k問題 → トンマナはA（思考）で、ビジネスレベルに連動。¥15kはLv1専用。
- ページ数多い問題 → B を「型×Lv＋量産」に。テンプレ展開で非線形。
- 参考有無問題 → Lvドライバーを「創出量×訴求密度×情報量」の3軸に。
