## 所見（料金・ポジショニング）

- **価格レンジは ¥3,000〜¥350,000**。「相談¥3,000 → コーディング¥15,000 → WP改修¥10,000 / サーバー代行¥5,000 → SEO運用¥200,000 → HP制作¥350,000」と、
  **入口（相談・小規模）から本命（HP制作・SEO運用）まで段階的に積み上がる導線**が組まれている。
- **売れ筋はコーディング（実績67）**が突出。HP制作（6）・SEO運用（6）・サポート（13）が続く。高単価のHP/SEOはまだ実績数は少なく、コーディングで信頼を貯めて上位メニューへ送客する構造。
- 認定：HP制作・SEO運用・コーディングが「PRO認定」、サポート・WP改修・サーバー代行が「プラチナ認定者」。
- 制作（Web制作/00）の A/B プラン（見やすく伝わる／売れる）に対し、ココナラ実出品は **HP制作＋SEO運用＋実装＋保守＋環境構築** に分解されており、Web制作/00 の「サービス範囲（作業範囲外：サーバー・ドメイン準備／運用代行）」とは**実出品の方が範囲が広い**（サーバー代行・SEO運用・保守をココナラでは単体出品している）。
