# 06. ブランディング・逆輸入戦略

> 出典：Notion「【2026/4/26】ビビドニア×ヨンマルヨン 逆輸入戦略・完全版」（2026/4/28 更新含む）
> 配信者ブランド「ビビドニア（白百合リリィ）」の集客を、制作サービス「ヨンマルヨン」へ
> **逆輸入**する戦略。価格競争ではなく**信頼ベース**で選ばれる状態を作る。

---

## 世界観の階層構造

```
クロノロク（代表・私）
  ↓ 公式見解・設計の最終権限・たまに出てくる人
  ↓ 「クロノロクが言ってることが正」という印象が定着済み
白百合リリィ（ビビドニア）
  ↓ 毎日22時配信・表の顔（ADHD設定の多少の暴走はクロノロクが上書きできる設計）
マオウ軍（クリエイター集団・設定）
  ↓ ヨンマルヨンの世界観母体
ヨンマルヨン（coconala・制作顔）
  → デジクリ部門の収益化窓口
```

> ゾーニングは完成済み。クロノロクがいつでも出動可能な状態を維持しているため、
> 白百合の暴走リスクは構造的に抑制済み。

---

## 逆輸入の実行イメージ（3フェーズ）

- **Phase 1：配信を盛り上げる**（エンタメエンジニアリング部門）
  - 配信ツール・ミニゲームを投入し、デジクリ部門に依存せず配信クオリティを上げる。
  - リスナーに「なんか進化してる」という印象を与える。
- **Phase 2：自然な接続**
  - 白百合が配信・Xで「制作してくれてるチームがいる」と言及。
  - マオウ軍・ヨンマルヨンへの自然な導線（「知り合いのクリエイターに頼む」感覚）。
- **Phase 3：ヨンマルヨンへの流入**
  - 同じVtuber活動者が「白百合が使ってるクリエイターチーム」として認知。
  - ココナラ上の「知らない人への不安」が解消され、**信頼ベース**で選ばれる。

### コンセプトの構造
```
404（よんまるよん）
= 見つからない、表に出ない
= あなたの裏方に
= 見えないけど、ちゃんとそこにいる
```

> **名前が弱いのではなく、coconalaページが止まっていることが自信のなさの正体。**
> コンセプトは完成している。受け皿（ココナラページ）を整備することが先決。

### コピーの課題
- 「あなたの裏方に」は方向性として正しいが、**キャッチコピーとしてはまだ弱い**。
- 404の意味（見つからない＝表に出ない裏方）をより鋭く言語化する必要あり（要ブレスト）。
