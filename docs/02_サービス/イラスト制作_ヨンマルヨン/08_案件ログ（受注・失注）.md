# 08. 案件ログ（受注・失注台帳）

> ヨンマルヨン（イラスト制作）の**案件ごとの結果を1行で蓄積**する台帳。
> 「どの価格帯・どの客層なら通るか／落ちるか」を後から見えるようにするのが目的。
>
> 運用ルール：
> - 見積もり提示・商談が発生した案件は、結果が出たらここに1行追加する。
> - クライアント名は匿名可（ハンドル・属性で可）。金額は税抜・総額で記載。
> - 詳細があるものは `records/` の該当MTG記録へリンクする。
> - 過去のココナラ販売実績（取得時点で総88件）は個別未記録。さかのぼり登録は任意。

## 集計・傾向（随時更新）

データが溜まったら、ここに「価格帯 × 客層 × 受注率」の傾向を書く。

### 現時点の所見（n=1・暫定）
- **個人Vtuber × 高単価（10万円超）は通りにくい**（抱き枕案件で実証）。
- 一方、ココナラ実績では **キャラクターイラスト ¥30,000 が主力（総88件中61件）** で回っている
  → 個人Vtuber層は**3万円前後の中単価帯**が受注ゾーンの可能性（要・追検証）。
  - 出典：`07_ココナラ現行出品ラインナップ（要実取得）.md`
- 高単価を狙うなら、ターゲットを**事務所・企業案件**へ寄せるのが定石
  （有望＝事務所裏方：イベント一括発注・20万でも払う・確実性/権利重視）。
  - 出典：`06_ブランディング・逆輸入戦略.md`、`04_量産セミオーダー企画.md`

### 集計の観点（テンプレ）
- 価格帯別の受注率（〜3万 / 3〜10万 / 10万〜）
- 客層別の受注率（個人Vtuber / 事務所・企業 / その他）
- 失注の主要因（価格 / 納期 / 仕様 / 連絡途絶 / その他）
- 受注に効いた要素（実績紹介 / 信頼文脈 / 価格 / スピード）
