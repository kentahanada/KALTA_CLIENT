# 00. ヨンマルヨン サービス概要

## 一言でいうと

**ヨンマルヨン（404）** は、株式会社カルタの**デジタルクリエイション部門**が運用する
**イラスト制作サービス**。展開プラットフォームは **ココナラ（coconala）**。

- コンセプト：**「404 = 見えない裏方」**（見つからない／表に出ない＝あなたの裏方に）
- 元々の主戦場：Vtuber向けイラスト・Live2D制作（エンタメ寄り）
- 現状：AI普及による市場縮小・価格破壊を受け、ピボット／再設計を進行中
- カルタとの関係：**アンダーグラウンド扱い**。カルタ公式のポートフォリオとしては扱わない

## ステータス

| 項目 | 内容 |
| --- | --- |
| 運用部門 | デジタルクリエイション部門 |
| プラットフォーム | ココナラ（coconala）。補助的に「つなぐ」やX運用を検討 |
| 事業ステータス | 縮小／再設計進行中（ピボット中） |
| ピボット先 | ①量産イラストアカウント（セミオーダー） ②デジタルコンテンツ制作サービスサイト（将来構想） |

## ポジショニング

スケブ（非同期・品質不安定・低単価）と、ハイエンド個人イラストレーター（高単価・属人的）の
**間**を埋める。「**安定品質・定価・確実納品**」を軸に、価格競争ではなく**信頼ベース**で選ばれる状態を目指す。

```
高単価・個人色強い                              低単価・使い捨て
   │                                              │
個人イラストレーター   ←【ここを狙う】→   スケブ・AI生成
（3〜10万円）         定価・安定品質・確実納品   （〜3,000円）
```

## ブランド構造（世界観）

ヨンマルヨンは、配信者ブランド「ビビドニア（白百合リリィ）」と連動した世界観の中で、
**制作の顔（収益化窓口）**として位置づけられている。詳細は
`06_ブランディング・逆輸入戦略.md` を参照。

```
クロノロク（代表）… 公式見解・最終権限
  └ 白百合リリィ（ビビドニア）… 配信・表の顔
      └ マオウ軍（クリエイター集団・設定）… 世界観母体
          └ ヨンマルヨン（ココナラ・制作顔）… デジクリ部門の収益化窓口
```
