# 05. 価格ルール（特急・大判・著作権）

> 標準料金表（`01_サービス一覧・料金表.md`）の枠を超える**非定型案件**に対する加算・方針ルール。
> 主出典：2026/6/8「ヨンマルヨン-抱き枕案件coconala再設計MTG」（→ `records/2026-06-08_抱き枕案件coconala再設計MTG.md`）

---

## 1. 特急料金

- 基準：従来の「2週間以内の納品」は**立ち絵等の標準的な作業**を想定したもの。
- 標準枠に収まる範囲での短納期対応は **×1.3倍**。
- 2週間の納期枠に収まらない非定型案件は、その都度**日数を考慮して個別判断**する。

| 区分 | 倍率 |
| --- | --- |
| 特急料金（短納期） | ×1.3 |

## 2. 大判サイズ制作費

- 対象：**超高解像度（目安 2万ピクセル以上）** の制作物。
  - 例：抱き枕カバー（6,890 × 24,803 px など）。
  - PC負荷が高く、線画・着彩等の工程で工数が大幅に増大するため、通常制作枠を超える。
- 位置づけ：オプションの中でも「**レベルマックス**」的な最上位加算。
- 加算：**×2.0倍**。

| 区分 | 倍率 |
| --- | --- |
| 大判サイズ制作費（2万px以上目安） | ×2.0 |

## 3. 見積もり計算式（加算の組み合わせ）

```
総額 = 制作費ベース × 特急料金倍率 × 大判サイズ倍率（×その他加算）
```

### 適用例（抱き枕案件・両面）
```
制作費ベース ¥95,000 × 特急 1.3 × 大判 2.0 ≒ ¥247,000
```
→ 非定型・大判の制作物に対する妥当な評価額として総額 **約24万7,000円** で決定。

## 4. 著作権・商用利用の方針

- **一律の金額設定はしない**。案件の状況に応じて**柔軟に対応**する。
- 権利関係はむしろ**制限を緩める方向**（商用利用・グッズ利用を許可する方針）。
- 標準料金表のオプション（グッズ利用許可 ¥10,000〜／著作権譲渡 全体費用の100%）は
  **目安**として残しつつ、実案件では内容に合わせて調整する。

> 🔧 関連タスク（MTG発）：商用利用・物理利用に関する著作権／権利関係ガイドラインを改定し、
> 案件内容に合わせた料金規定を整理する（担当：花田）。

## 5. 見積書の記載方針（運用）

- 見積書・資料には**内訳（打ち分け）を記載しない**。**納期・進行状況・納品**に関する項目のみ記載。
- 見積もり根拠・打ち分けの質問には資料上で回答せず、担当（花田）が個別対応する。
- 納期条件として「最短納期目安」「確認作業は24時間以内」である旨を明記する。
- 購入が遅れて納期がひっ迫した場合は、スケジュール調整で対応（担当：花田）。
