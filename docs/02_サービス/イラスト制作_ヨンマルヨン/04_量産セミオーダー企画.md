# 04. 量産イラストセミオーダー企画（v2.0）

> 出典：Notion「量産イラストセミオーダー企画書 v2.0」（親：※凍結-量産イラスト企画）
> ⚠️ 親ページが「凍結」ステータス。本企画は**凍結／保留中**の可能性があるため、再開時は最新の判断を要確認。

ヨンマルヨンのピボット方向の一つ。**「安定品質・定価・確実納品」のセミオーダー専門サービス**として、
Vtuber・個人クリエイター向けにイラストを量産受注する事業構想。

---

## 1. ターゲット定義

**ターゲット像**
- 個人VTuber（登録者1,000〜5万人規模）
- SNSでオリキャラを運用しているイラスト系ユーザー
- FANBOX等でコンテンツ販売をしている人
- 3Dアバター持ちで2D差分も欲しい層

**ターゲットが求めること**
- 自分のキャラを安価でイラスト化したい（SNS投稿・壁紙・限定コンテンツ用）
- スケブより安定・確実・早い
- 差分で頼むので成果物がわかりやすい

**ベースキャラの方向性**
- サブカル系ファッション（現VTuber界隈で受けるトーン）。営業担当が収集した参考アカウントリストで確定。
- パステル・セーラームーン系は看板にしない。

## 2. 価格設定・メニュー構造

### ベースプラン
| メニュー | 内容 | 価格（税込） |
| --- | --- | --- |
| セミオーダーS | バストアップ・表情1種・背景なし | 6,000円 |
| セミオーダーM | バストアップ・表情3種・背景なし | 9,000円 |
| セミオーダーL | 全身・表情3種・背景なし | 14,000円 |

### オプション
| オプション | 価格 |
| --- | --- |
| シンプル背景追加 | +3,000円 |
| 洋服カラー変更 | +1,000円／着 |
| 急ぎ対応（通常納期の半分） | +3,000円 |
| データ形式追加（PSD等） | +1,500円 |

### 修正ルール
- **修正1回まで無料**（ラフ確認後の軽微な修正に限る：髪の細部調整・表情微修正・色変更など）。
- 含まない（追加料金）：ポーズ変更・キャラデザの大幅変更・背景追加。
- 追加修正：1回あたり 1,500円〜（内容による）。
- ラフ時に大幅な方向修正が必要な場合は「発注情報の不足」として扱い、再ヒアリング。

## 3. マーケティング戦略

### ポジショニング
スケブ（〜3,000円・品質不安定）とハイエンド個人イラストレーター（3〜10万円）の**間**を、
定価6,000〜14,000円・安定品質・確実納品で狙う。

### 差別化ポイント
- 発注しやすい（フォーマットが整っている）
- 納期が読める（標準10日以内・急ぎ5日以内）
- 品質が安定（ディレクターが発注を設計し大外ししない）
- 修正1回無料（安心感の担保）

### KPI・フェーズ設計
| フェーズ | 期間 | 目標 | 次フェーズ条件 |
| --- | --- | --- | --- |
| 1 信用構築 | 2〜3週間 | WIP投稿20件以上・フォロワー100人 | 投稿が安定継続 |
| 2 初期受注 | 〜1ヶ月 | 受注3件 | 3件完納・クレームなし |
| 3 拡大 | 〜3ヶ月 | 月10件・月商6万円以上 | リピーター発生・実績投稿5件以上 |
| 安定期 | 3ヶ月〜 | 月20件・月商15万円以上 | プラットフォーム展開・定期契約開始 |

### チャネル戦略
| チャネル | 役割 | 開始 | 判断基準 |
| --- | --- | --- | --- |
| X専用アカウント | メイン集客・ブランディング | フェーズ1 | 即時開始 |
| ピクメニアカウント | 既存フォロワーへの拡散 | フェーズ2 | 募集に合わせRT |
| 「つなぐ」 | 検索流入（手数料5.5%） | フェーズ2 | 実績3件後に登録 |
| ココナラ | 検索流入（手数料22%） | フェーズ3 | 月10件安定後に判断 |

## 4. 制作フロー（要点）

```
集客・告知 → 受注（フォーム/DM） → 受付確認（24h以内・テンプレ）
 → 発注整理（発注シート） → ラフ制作（受注から3日以内）
 → ラフ確認（依頼者へ共有・修正1回無料） → 修正対応
 → 仕上げ（ラフ確認から5日以内） → 納品 → アフター（感想・実績投稿）
```
標準納期：受注から10日以内 ／ 急ぎオプション：5日以内（部門別フローは `02_制作作業フロー.md` 参照）

### 発注シートの必須項目
- キャラクター名・性別・年齢感
- 髪型（参考画像 or 文字説明）
- 服装系統（参考画像3点以上 + ジャンル名）
- ポーズ（参考画像 or 希望）
- 表情差分の種類と枚数
- 体型・肌色・目の色
- 背景の有無・内容
- 納期・プラン・オプション内容

### ツール構成
Asana（タスク・進捗）／ Geminiジェム（DM対応テンプレ）／ Google Drive（発注シート・参考画像）／
X専用アカウント（集客・受付）／「つなぐ」（プラットフォーム受注）

---

## 5. リスクと対策（抜粋）
| リスク | 対策 |
| --- | --- |
| クレーム・大幅修正 | 発注シート必須項目の徹底・ラフ確認を必須化 |
| 制作遅延 | 同時進行は最大3件・Asanaで日次確認 |
| 修正過多 | 受付時にフォームで具体化・2回目以降は有料明示 |
| 集客不発 | フェーズ1（WIP投稿）を必ず先行・1ツイートに賭けない |
| キャラ大外し | 参考画像を3ジャンル分け・テンプレを埋めてから渡す |
| 受注過多でパンク | フェーズ2は受注上限 月5件・慣れてから拡大 |
