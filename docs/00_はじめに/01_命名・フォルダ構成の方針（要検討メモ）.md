# 01. 命名・フォルダ構成の方針（要検討メモ）

> このドキュメントは「**まだ確定していない／今後判断したい**」命名上の論点を集約するものです。
> 決め切らずに保留している事項を明示することで、将来の意思決定を楽にします。

## 背景

ヨンマルヨンはデジタルクリエイション部門のイラスト制作サービスとして、現在ココナラで展開中です。
一方で、将来的にカルタの **「デジタルコンテンツ制作サービス」** へと昇華する未来もあり得ると
考えています。ただし、

- ヨンマルヨンの現在のユーザー属性（Vtuber・個人クリエイター等、エンタメ寄り）と、
- toB向け高額路線の「デジタルコンテンツ制作サービス」が狙う層

には差があり、**そのまま地続きにするのは難しい側面**があります。
そのため「今すぐその未来を見据えて設計しすぎる必要はない」が、「認識としては持っておきたい」
という温度感です。 → **フォルダ名・カテゴリ名は要検討。**

## 現時点の暫定方針

| 項目 | 暫定の扱い | 理由 |
| --- | --- | --- |
| サービス枠のフォルダ名 | `02_サービス/イラスト制作_ヨンマルヨン/` | **現状に忠実**な名前を採用。今やっているのは「イラスト制作」 |
| 「デジタルコンテンツ制作」 | `03_将来構想/` に**構想として分離** | 現サービスと地続きにせず、別の枝（superset）として保留 |
| 共通の根っこ | `01_共通基盤/` | Web/イラスト/将来のデジコンすべてに効く土台 |

## 決定事項（2026-06-08）

- **構成は現状維持（横並び据え置き）で確定**。`イラスト制作_ヨンマルヨン` と `web制作` を
  `02_サービス/` 配下に横並びで置き、「デジタルコンテンツ制作」は `03_将来構想/` に分離したまま。
  → 2本柱への即時再編・リネームは**行わない**。
- **Web制作はデジタルコンテンツ（イラスト）より一歩先**にいる状態。理由：
  - ココナラ展開に加え、KALTA自社ブランド **「ピクセルスタジオ」（pxstudio.jp）** での
    サービス展開が進んでいる（詳細・法人関係は要確認 → `02_サービス/web制作/00_サービス概要.md`）。
- 本リポジトリのスコープは**クライアントワーク特化**。社内プロダクト（IP/ゲーム等）は対象外。
