# KALTA デジタルクリエイション ナレッジリポジトリ

株式会社カルタ（KALTA）の **クライアントワーク系サービス** に関する設計思想・業務フロー・
料金体系・対応マニュアル・MTG記録などを集約・体系化するためのリポジトリです。

現在の主対象は、デジタルクリエイション部門がココナラ（coconala）で展開している
イラスト制作サービス **「ヨンマルヨン（404）」** です。あわせて、同じ設計思想を共有する
姉妹サービスである **Web制作サービス** との住み分けも記録します。

---

## 📐 基本の考え方

> **根っこにある基本の設計思想は、Web制作もデジタルコンテンツ（イラスト）制作も同じ。**
> そこから先の「サービス固有の詳細」がそれぞれ枝分かれしていく。

このリポジトリは上記の思想を構造として表現します。

```
共通の根っこ（設計思想・業務フロー・見積もり設計フレーム）
   ├─ サービス別の枝：イラスト制作（ヨンマルヨン）
   └─ サービス別の枝：Web制作
        └─ （将来の枝：デジタルコンテンツ制作サービスへの昇華 … 構想段階）
```

---

## 🗂️ ディレクトリ構成

| パス | 内容 |
| --- | --- |
| `docs/00_はじめに/` | リポジトリの読み方・命名方針（**フォルダ名は要検討事項あり**） |
| `docs/01_共通基盤/` | 全サービス共通の設計思想・業務フロー・見積もり設計・調整費ルール |
| `docs/02_サービス/イラスト制作_ヨンマルヨン/` | ヨンマルヨンのサービス概要・料金表・制作フロー・SOP・価格ルール・ブランディング・MTG記録 |
| `docs/02_サービス/web制作/` | Web制作サービスの概要（住み分け参照用） |
| `docs/03_将来構想/` | 「デジタルコンテンツ制作サービス」への昇華構想（構想段階） |

---

## 🔗 主な情報源

このリポジトリは以下を一次情報として整理しています。

- Google ドライブ：対応マニュアル類・料金表・MTG文字起こし／要約
- Notion：ココナラ関連ドキュメント群（共通業務フロー、サービス別モジュール、SOP、企画書ほか）
- 各種MTGログ・過去ログ

> ⚠️ 一次情報（Notion / Google Drive）が正、本リポジトリは「整理・体系化したスナップショット」です。
> 数値や運用ルールに差異が出た場合は一次情報を確認のうえ更新してください。
